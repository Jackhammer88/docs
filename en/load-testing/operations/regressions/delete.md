# Deleting a dashboard

If you no longer need a dashboard, delete it.

{% list tabs group=instructions %}

- Management console {#console}

  1. In the [management console]({{ link-console-main }}), select **{{ ui-key.yacloud.iam.folder.dashboard.label_load-testing }}**.
  1. In the left-hand panel, select ![image](../../../_assets/load-testing/regressions.svg) **{{ ui-key.yacloud.load-testing.label_regressions-title }}**.
  1. To the right of the dashboard name, click ![image](../../../_assets/options.svg) and select **{{ ui-key.yc-ui-datasphere.common.delete }}**.

{% endlist %}
