# Посмотреть группы пользователя

Вы можете посмотреть, в какие [группы](../../organization/concepts/groups.md) входит конкретный [пользователь](../../overview/roles-and-resources.md#users).

{% list tabs group=instructions %}

- Интерфейс {{ cloud-center }} {#cloud-center}

  1. Перейдите в сервис [{{ org-full-name }}]({{ link-org-cloud-center }}).
  1. На панели слева выберите ![icon-users](../../_assets/console-icons/person.svg) **{{ ui-key.yacloud_org.pages.users }}**.
  1. В списке пользователей выберите нужного пользователя.
  1. Перейдите на вкладку **{{ ui-key.yacloud_org.user.title_groups }}**.

{% endlist %}
