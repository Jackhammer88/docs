По умолчанию в подключение добавлены ключи с именем и идентификатором оператора и клиента. В поле **{{ ui-key.yc-ui-talkanalytics.connections.column.name }}** введите название, под которым ключ будет отображаться в {{ speechsense-name }}. При необходимости добавьте описание.

Чтобы указать дополнительные метаданные для оператора и клиента, нажмите кнопку **{{ ui-key.yc-ui-talkanalytics.connections.add-key }}**. Для нового ключа обязательно введите его название и выберите [тип метаданных](../../../speechsense/concepts/resources-hierarchy.md#connection).
