# Изменить способ оплаты

Изменить способ оплаты можно только в платежном аккаунте с типом **{{ ui-key.yacloud_billing.billing.account.create-new.account-type_label_company }}**. 

## Сменить оплату банковской картой на перевод с расчетного счета {#card-to-transfer}

Чтобы изменить способ оплаты:

1. Составьте запрос на изменение способа оплаты.

   {% include [change-payment-docs](../_includes/change-payment-docs.md) %}

1. Отправьте запрос на электронную почту [{{ billing-docs-email }}](mailto:{{ billing-docs-email }}). Запрос рассматривается в течение трех рабочих дней. Менеджер {{ yandex-cloud }} сообщит вам о принятом решении.

{% note info %}

После того как ваша карта будет отвязана от платежного аккаунта, вы не сможете привязать новую карту. [Пополнять лицевой счет](../operations/pay-the-bill.md) можно будет только с помощью [перевода средств с расчетного счета (через банк)](../payment/payment-methods-business.md).

{% endnote %}

## Сменить оплату переводом с расчетного счета на банковскую карту {#transfer-to-card}

Чтобы изменить способ оплаты, отправьте запрос на смену способа оплаты на [{{ billing-docs-email }}](mailto:{{ billing-docs-email }}) и укажите [идентификатор](../concepts/billing-account.md#billing-account-id) вашего платежного аккаунта. Запрос рассматривается в течение трех рабочих дней. Менеджер {{ yandex-cloud }} сообщит вам о принятом решении.
Подробную информацию о правилах оплаты банковской картой см. в [описании методов оплаты для юридических лиц](../payment/payment-methods-card-business.md).
