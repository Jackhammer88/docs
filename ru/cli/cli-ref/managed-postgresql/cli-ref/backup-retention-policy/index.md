# yc managed-postgresql backup-retention-policy

Manage PostgreSQL backup long-term retention policies

#### Command Usage

Syntax: 

`yc managed-postgresql backup-retention-policy <command>`

Aliases: 

- `backup-retention-policy`
- `backup-retention-policies`

#### Command Tree

- [yc managed-postgresql backup-retention-policy create](create.md) — Create new backup retention policy
- [yc managed-postgresql backup-retention-policy delete](delete.md) — Delete specific backup retention policy
- [yc managed-postgresql backup-retention-policy list](list.md) — List available backup retention policies
