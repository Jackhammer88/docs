# yc quota-manager quota-limit

Manage quota limits

#### Command Usage

Syntax: 

`yc quota-manager quota-limit <command>`

#### Command Tree

- [yc quota-manager quota-limit get](get.md) — Show information about the specified quota
- [yc quota-manager quota-limit list](list.md) — List quota limits
- [yc quota-manager quota-limit list-services](list-services.md) — List quota services
